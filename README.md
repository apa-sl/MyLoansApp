# My Loans App
A final project for CS50X course from HarvardX.  

## App purpose
Track all personal loans repayments (how much installemts there are to be paid each month, store history of the loans).  

## deployment
1. Rename _configTEMPLATE.ini_ to _config.ini_
2. Edit _config.ini_ and put MySQL access credentials

## Tech stack
- Database: MySQL
- Backend: Python + Flask
- Frontend: Bootstrap
